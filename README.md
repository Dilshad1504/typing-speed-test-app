# Typing Speed Test App

A web-based typing speed test application that calculates typing speed (WPM) and accuracy in real-time.

## 🚀 Features
- Real-time typing speed (WPM) calculation
- Accuracy tracking
- Timer-based typing test
- Dynamic text generation using API
- Reset functionality
- Clean and responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript
